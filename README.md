
# SDDM Theme for SteamFork

Maintained as a solution for desktop mode of [SteamFork](https://github.com/steamfork).

> [!IMPORTANT]
> This theme is suitable for Plasma 6.

## Dependencies
* sddm
* qt6-5compat

## Installing
```
sudo make install
```

## Packaging
```
make package
```
